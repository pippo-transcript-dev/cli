# Examples

This directory is intentionally kept free of private documents.

Good public examples should be synthetic or explicitly licensed for redistribution:

- a small one-page PDF with two text columns;
- a synthetic receipt image;
- a synthetic business card image;
- a simple table PDF;
- a simple chart image.

Generated outputs should not be committed. Use a temporary output directory:

```bash
pippo-transcript examples --output sample-output
```
